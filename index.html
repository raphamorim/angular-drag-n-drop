Get This!

Download Drag'n Drop!
$ bower install angular-drag-n-drop

Basic Usage

Add the angular drag'n drop source on body end:

<script src="angular-dragdrop.min.js"></script>
</body>

Start the Drag'n Drop in any element:

<div ng-app="dragAndDrop" ng-controller="dragAndDropController">

For each element allow dragging, use draggable directive:

<p draggable>Indiana Jones</p>
<div draggable>Obi Wan Kenobi</div>

For allow drop in any element, use droppable directive:

<div droppable class="box"></div>
